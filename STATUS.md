# Status

This repository is a portfolio preview and intentionally incomplete. The goal is to demonstrate architecture, data modeling, and end-to-end flows, not a production-ready planner.

## Implemented

- Calendar view with month navigation
- Tasks and subtasks with completion tracking
- Statistics summaries
- JSON/CSV export

## Planned (not yet implemented)

- Recurring tasks
- Authentication and multi-user support
- Tests and CI
- More robust validation and error handling
- Accessibility improvements
